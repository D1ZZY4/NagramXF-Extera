# Features

Nagram Extera inherits everything the upstream Telegram client offers, plus
the long list of additions accumulated by the Nagram X / XF lineage and a
selection of features ported from sister forks.

The list below is intentionally short; it focuses on what is *meaningfully
different* from stock Telegram. For the exhaustive per-setting reference,
open the in-app **Settings → Nagram** screen.

## Privacy & anti-tracking

- **Save deleted messages** — keep a local copy of messages you have already
  read when the sender deletes them.
- **Save edits history** — see every edit a message has gone through.
- **No "Last seen" leaks** — fine-grained controls beyond what stock Telegram
  exposes.
- **Hide phone number** in the side menu and account switcher.
- **Disable Google services lookup** for proxies, sticker suggestions, etc.

## Messaging power-tools

- **Translation** with multiple providers (Google, DeepL, LibreTranslate,
  Yandex, system, and on-device ML Kit).
- **Forward without quote / without caption** as defaults.
- **Repeat message** N times (useful for stickers and reactions).
- **Send message as channel** quick toggle.
- **Per-chat unique font, wallpaper and bubble style.**
- **Click-to-copy** any message text without opening the context menu.

## Chats list

- **Folder tabs** with custom ordering, hide-on-scroll, and per-folder
  unread badges.
- **Pin unlimited chats** in folders.
- **Hide archive automatically** when empty.
- **Compact / classic chat list densities.**

## Media

- **Stable Stories disable** — fully hide the Stories row and notifications.
- **Sticker size slider**, including animated and video stickers.
- **Save GIFs / videos / voice notes** to a custom folder.
- **Voice-to-text** via on-device speech recognition where available.

## Misc

- **Premium-style features unlocked** for cosmetic options that don't
  require Telegram-side entitlements (icon packs, app icons, reaction
  styles).
- **Custom proxies UI** with import/export from `tg://` links and `.json`.
- **Built-in MTProto-over-WebSocket / Trojan / Shadowsocks / V2Ray
  proxy support** carried over from the Nagram lineage.
- **Disable in-app browser** in favour of the system default.

## What we deliberately do **not** add

- No analytics, no telemetry, no opt-in "anonymous usage" pings.
- No cryptocurrency, ads, or referral systems.
- No "AI assistant" features that send your message contents to a
  third-party LLM by default.
